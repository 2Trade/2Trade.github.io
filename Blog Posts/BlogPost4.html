Firebase Chatroom

Today we used Firebase to try and make a chatroom we could put on our website. We learned that making a simple chatroom would take a lot more time and effort than what we previously expected. Firebase relies heavily on terminal/sublime text, and also requires the downloading of many different applications to work. Everything from the creation of a user sign-in system to the "send message" system, every part took much more effort than we expected when we were told to make this.

Firebase tutorial: How to make a Friendly Chatroom using Firebase

Link to tutorial: https://codelabs.developers.google.com/codelabs/firebase-web/#0

1. Set up the chat.
Before doing anything, make sure all of the sample code is in place, and install Firebase CLI(Command Line Interface).
2. Add Firebase to the app.
Go to the Firebase console, click "Create New Project", then click "Add Firebase to your Web App" copy down the code, then put it into your app's console.
3. Test-Run the App
Now that Firebase is on the app, test it out. Run this code:
$web-start> firebase serve
Listening at http://localhost:5000
4. Designing Sign-In.
No one can use your app if it doesn't have any sort of sign-in feature! Type this code:
FriendlyChat.prototype.initFirebase = function() {
this.auth = firebase.auth();
this.database = firebase.database();
this.storage = firebase.storage();
this.auth.onAuthStateChanged(this.onAuthStateChanged.bind(this));
}
5. Import JSON.
Import the JSON file by going to Database and then clicking Import JSON.
6. (Optional) Set Up Security.
To keep your site safe, set up a simple security system. This primarily uses GitHub.
7. Send Messages.
People need to be able to send messages for it to be a chatroom. The code consists of mostly if/else statements in Sublime Text/Python.
8. Send Images.
This is very similar to the Send Images step, but use ".saveImageMessage" instead of ".saveMessage".
9. Deploy App.
Use Firebase static hosting to deploy the app.
10. Congrats!
You are done.
