10 Step Tutorial On What We Learned Today In Python

1. Define Your Function
Use def in Python to assign a function to a name of your choice:
def func(num1)->float:
Whatever you put underneath that will be the function. The "float" just means that it will output a float.
2. Decide What To Put in the Function
My personal function added one to a given integer. But you can put anything in here! (Not really everything but a lot of stuff). Since I had to make an entire list go through the list, I used a for loop inside my function to enable it to go through the entire list. After that, the function returns the data and I print the function's results. Make sure that you always end your function with
return num1
3. Use Your Function
Now that you've defined your function, you can call it whenever you want! Say your function was to add two numbers. Now, whenever you want, you can us it to add two numbers using your new add function.
4. Setup Tkinter Graphics in Python
When you do this, at the top of your page should be import tkinter. Without it, the following will not work. Underneath the first line, write this:
window = tkinter.Tk()
canvas = tkinter.Canvas(window, width=500, height=500)
canvas.pack()
Now at the bottom of the page, write this:
tkinter.mainloop()
In between the second to last and last line is where we are going to draw in tkinter graphics
5. Coordinates
Before we start drawing, we have to know how coordinates work inside python. It's a little different from normal mathematics. The x-axis lies at the top of the page while the y-axis is to the left. The units are in pixels. The amount of pixels on the canvas is set in step 4 to 500x500. This will prove to be very important.
6. Drawing a Line
Lets start by drawing a line. We do this like so:
canvas.create_line(w, x, y, z)
Now with w, x, y, and z in the parentheses, nothing will happen. That's because this is where the coordinates go. The first two coords being the x/y values for one endpoint on the line. While the last two coords are the x/y values for the second endpoint. Try inputing some values for the coordinates. (Don't input numbers that go above the range that you set in step 4 or the line will go offscreen)
7. Making a Rectangle
First type the following:
canvas.create_rectangle(w, x, y, z)
The letters mean the exact same thing as they did in the previous step so nothing new there. The cool thing is that to make a perfect rectangle, you only need two coordinates. This means that the formatting for making a rectangle is the same as making a line! So easy!
8. Combining Designs
Now that you've learned how to make lines and rectangles in python, you could probably use that to create some pretty cool designs! You're only limited to your own imagination!
